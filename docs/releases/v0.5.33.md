# Release preparation: next patch candidate

This release contains the completed Trust Layer remediation sequence:

- enforced Preflight gate by default;
- raw request, declared intent, and capability binding;
- multilingual and hidden-risk boundary coverage;
- template/adopter stability evidence;
- enterprise security and compliance boundary documentation; and
- the expanded offline “knows when to stop” demo.

Release source identity, SBOM, provenance, digests, smoke, and compatibility evidence are produced by the release workflow. The candidate does not claim enterprise security certification or runtime isolation.
