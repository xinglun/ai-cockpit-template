# v0.5.40

v0.5.40 is the corrective release after the prior release was quarantined when public Quick Install verification found that its immutable tag contained an installer whose digest did not match `release.json`. The release finalizer now regenerates `installerDigest` from the candidate installer, release preflight independently rehashes `install.sh` from the exact selected source commit, and preparation-mode distribution checks reject installer drift before any tag or GitHub Release mutation. The historical tag is not rewritten; v0.5.40 becomes public only after its Work Item PR is merged and closed, exact-main smoke and compatibility pass, the workflow verifies the source-bound archive and installer, and the public distribution check succeeds.
