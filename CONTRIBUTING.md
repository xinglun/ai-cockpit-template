# Contributing

This repository is a template for governance workflows, checks, and release
evidence.

Contributions should:

- Preserve the template nature of the repository.
- Prefer small, mechanically verifiable changes.
- Update or add tests when changing behavior.
- Keep documentation, checks, and generated evidence in sync.

Before opening a change:

1. Run the relevant make targets or tests for the area you changed.
2. Verify that the AI Cockpit checks still pass.
3. Avoid introducing repository-specific assumptions unless they are clearly
   marked as examples.
