# v0.5.41
v0.5.41 repairs the publication-state gap found after the immutable v0.5.40 attempt. Release finalization now records SHA-256 archive verification as `verified` only when it has generated the canonical archive and installer bindings. After creating the exact tag and Draft assets, the release workflow runs the real tagged Quick Install verifier before it may change the Draft to public.
The quarantined v0.5.39 and v0.5.40 tags remain unchanged and their GitHub Releases remain Draft. v0.5.41 is published only after its governed PR lifecycle, merged-main CI, exact-source preflight, tagged prepublication verification, and independent public distribution audit all succeed.
